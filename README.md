# e-plantShopping

## Project Overview

**e-plantShopping** is a modern e-commerce web application for purchasing plants online. This application provides users with an intuitive shopping experience where they can browse through a variety of plants, add items to their cart, and manage their purchases efficiently.

## Features

- **Product Catalog**: Browse through a diverse collection of plants with detailed information
- **Shopping Cart**: Add, remove, and manage items in your cart with real-time updates
- **State Management**: Utilizes Redux Toolkit for efficient state management
- **Responsive Design**: Mobile-friendly interface that works seamlessly across all devices
- **About Us**: Learn more about our mission and commitment to providing quality plants

## Technologies Used

- **React**: Front-end library for building the user interface
- **Redux Toolkit**: State management solution
- **Vite**: Build tool for fast development and optimized production builds
- **CSS**: Custom styling for an attractive user experience
